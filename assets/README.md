# Assets Folder

Place your app assets in this folder:

## Required Files

1. **icon.png** (1024x1024)
   - Main app icon
   - Should be a square image
   - Will be used for iOS and Android

2. **splash.png** (2048x2048 or larger)
   - Splash screen shown when app launches
   - Should have important content in the center
   - Background color: #f97316 (orange)

3. **adaptive-icon.png** (1024x1024)
   - Android adaptive icon
   - Foreground layer for Android
   - Should be centered with transparency

4. **favicon.png** (48x48 or larger)
   - Used for web version
   - Small icon for browser tab

## Quick Setup

You can use online tools to generate these:
- [App Icon Generator](https://www.appicon.co/)
- [Expo Icon Generator](https://icon.kitchen/)

Or create placeholder images temporarily for development.

## Example Icon Colors

For the "Plan B" bar finder app, consider:
- Primary: Orange (#f97316)
- Background: White or dark
- Icon: Beer mug, cocktail glass, or map pin
